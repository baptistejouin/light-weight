# Stack

This project was started as a side project to learn SolidJS, very similar to React by design but with some design changes made for performance and a better development experience, so I just wanted to give it a try. I also need a gym app tailored for my needs, so it's a solid decision.

This project is volontary mobile first and will be a PWA

## Developing
```bash
pnpm i
pnpm run dev
```
